Github图床

简单说图床就是一个在网络上存储图片的地方，目的是为了节省本地服务器空间，加快图片打开速度，主要是个人博客和网站使用

我这里主要说的是github图床 现在还有一个工具—PicGo，但这里不作解释，感兴趣的朋友可以自行前往查看

  1. 新建一个repositories

  2. 进入repositories下的wiki选项中

  3. 复制wiki对应的克隆地址

  4. 在本地克隆项目

$ git clone https://github.com/你的github用户名/你的图床仓库名.wiki.git

这之后会生成一个文件夹，目录看Git上的提示

  1. 进入文件夹，将要上传的图片放在文件夹中

  2. 打开命令行，依次输入命令:

    $ git add photo.png

    $ git commit -a -m "update"

    $ git push
  1. 现在已经将图片放置Github图床，图片的地址为： https://raw.githubusercontent.com/wiki/你的github用户名/你的图床仓库名/photo.png
