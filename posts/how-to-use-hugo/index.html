强大的静态blog工具-Hugo

Hugo 是一个基于 Go 语言开发的静态网站生成器（SSG），目前由 @bep 领衔开发，Hugo 的突出特点是简洁、灵活、高效，目前有很多知名网站都在使用 Hugo：Netlify、Let’s Encrypt、IPFS、Cloudflare Developers、DigitalOcean Docs、1Password 等等。与目前国内流行的 Hexo 相比，Hugo 的速度可称为飞速?——在安装和使用上都是如此

在此之前我们需要先安装好Git分布式版本管理工具，这个工具可以很好的帮我们部署网站和上传后续的更新，安装git请看廖雪峰大神无比详细的教程

那就让我们开始搭建属于自己的hugoBlog吧~~~

先让我们来到hugo的官方网站

接着点击首页的quick start(快速开始)

这时候会导向hugo的安装教程,官方的教程还是挺详细的，还有视频，但是有几个点要注意一下的：

  1. 下载后解压到某个盘(不一定要在C盘内)，但路径一定不能出现中文，否则会有意想不到的问题出现

  2. 接下来要配置hugo的环境变量，不然hugo会无法运行(环境变量的配置教程自行百度吧，不同系统会存在部分差别，根据自己的系统进行配置)

  3. 环境变量配置好了，打开命令行窗口,输入hugo -h，出现如下图就代表我们的hugo环境配置好了

这时候你可以使用hugo来建立一个站点了！

这是hugo的中文文档也很详细哦~可以参照这里进行配置 比如说你的hugo放在D盘下，我这有两种办法来建立新站点，效果一样~

  1. 进入在D盘的hugo文件夹,如下图所示：

  2. 或者你使用命令符(同时按下WIN+R调出命令行) cd 你要进入的文件夹

操作完成后输入命令:hugo new site 你想创建站点的名字(别用中文...)如下图所示：

至此，这样你就创建了一个属于自己的站点了~

接下来你需要一款好用的编辑工具，我在此推荐微软的VisualStudioCode(VScode) 你还可以在VScode里添加markdown工具，提升你撰写文章的效率(添加工具自己在软件内探索，我在此不再赘述啦~)

接下来就开始正式创作你的blog了~~~

接下来我们进入hugo官方网站

  1. 找到Themes一栏
  2. 一开始咱们先来个简单点的主题吧~Hugo Bootstrap v4 Blog
  3. 点击进入下载后如图：
  4. 下载.zip压缩包，解压到如图所示位置，并删除后缀-master： 这里注意不要文件夹套娃哦~
  5. 进入exampleSite文件夹
  6. 复制里面的所有文件
  7. 粘贴到你的站点根目录下(有提示有重复是否替换的全部选择是)
  8. 替换完成后，双击打开config.toml(配置文件)
  9. 我们在VScode里编辑一下配置文件吧~

baseurl = “https://example.org/" 这是你以后网站的链接，比如说你寄管在Github上，到时候记得修改这里 languageCode = “en-us” 这个是配置语言环境，我们就可以修改成中文环境"zh-CN” title = “Hugo Bootstrap v4 Blog"这个就是你这个网站的名字啦 theme = “hugo-theme-bootstrap4-blog"这项不需要改动

接下来的就是主页显示多少篇文章；预览多少字数；还要作者信息；侧边栏配置之类的，只要你熟用有道翻译，都是可以轻松使用上滴！这些就留给你自己探索啦~

posts文件夹中的是你的文章，想修改或者删除在这里进行，但是写一篇新文章你得往下看

我们在站点的根目录下打开命令行，什么是根目录什么是命令行往上拉回去看

我们在这敲下命令 hugo new posts/你的文章名字.md

posts/是我们在posts文件夹下创建文章，这里的就是你正常的文章，但是如果不在posts/下创建会怎么样呢，往下看 提示创建成功以后打开文件夹你会看到刚刚新创建的markdown文件 title: “The First Blog” 这个是这篇文章的标题 date: 2020-02-10T13:59:23+08:00 日期是根据你敲下命令行的瞬间记录下的 draft: true 这个若为true代表为草稿，我们让他变为文章需要改为false 注：上下各有的”—“不要手贱删除了啊 写好你的内容以后我们还是在站点的根目录打开命令行，敲下命令 hugo server 看到链接了吗(http://localhost:50420/) 你的可能和我不一样，但这就是你的个人站点了，不过这只是一个只能在你这台电脑上访问的站点了，这时候hugo的优势也体现出来了，一个是生成速度快，另外一个是修改markdown文件可以实时修改网站，只需要在VScode里按下ctrl+s保存内容即可 接下来还是在站点根目录下运行命令行 hugo 这时候你再回去看你的站点文件夹，你会发现多了一个public文件夹，这里面就是这个工具根据主题生成的html(网页)文件，接下来只需要把这个文件夹里的内容全部上传到网上即可，这里我推荐寄管在github下，这样是免费的，比较适合个人博客，github+hugo的教程我会在下次上传，Enjoy~~~

再复习一下步骤吧： 首先我们创建站点 再把下载好的主题放入themes文件夹当中，记得去掉后缀名 然后把示例站点中的文件复制粘贴到站点根目录下，全部选择替换 接着修改相关配置文件 开始编辑你的文章 最后使用hugo命令生成public文件夹，文件夹内的就是网站文件了 最最后上传到服务器或者托管到其他网站(github+hugo教程下次上传) 这就大功告成啦
